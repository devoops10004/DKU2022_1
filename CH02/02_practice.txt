02. 실습

ⓐ 깃허브에 기존 동기화된 폴더에 CH02 하위폴더 추가하기. 또는 원격 저장소를 추가한 후 로컬 폴더에 동기화 하기
ⓑ 파이썬 IDLE 실행하기
ⓒ 파이썬 script 편집기 실행하기
ⓓ PDF 교재의 프롬프트 예제를 script 편집기에 타이핑하기
ⓔ 타이핑한 코드를 ⓐ에서 만든 폴더에 CH02-페이지수.py 파일로 저장하기
   예를 들어 교재의 47페이지를 코딩한 파일은 CH02-47.py로 저장
ⓕ run 시켜보기 <- 메뉴의 'run -> run module'을 클릭하거나, 키보드의 'F5'키를 누름
ⓖ 소스트리 프로그램을 실행하여 지금까지 편집한 폴더와 파일을 add, commit, push하여 깃허브에 등록하기
ⓗ 마지막으로 저장소 위치를 복사하여 교수 이메일로 보내기 (jsun1u@gmail.com)
